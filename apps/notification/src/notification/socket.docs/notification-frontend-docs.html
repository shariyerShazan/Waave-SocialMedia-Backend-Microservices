Real-time Outbound Delivery WebSockets (Socket.io)

Establish a persistent connection and subscribe to the channels described below to capture server pushes instantly.

Target Event Channel Direction Internal Provider
notification Server → Client Notification Service Gateway Pub/Sub 
// Client Subscription Hook Example
socket.on('notification', (payload) => {
  console.log('Received active notification:', payload);
  showToastNotification({
    title: payload.title,
    body: payload.body,
    avatar: payload.sender?.avatar
  });
  updateBadgeCounter(payload.unreadCount);
});

HTTP Endpoint Inventory REST API Gateway

API Base URL: http://localhost:4000

GET /notifications

Retrieve paginated list of alerts. Queries: page, limit.

PATCH /notifications/:id/read

Mark a targeted record as read.

PATCH /notifications/read-all

Mark every unread alert belonging to the current user as read.

DELETE /notifications/:id

Delete/dismiss an alert record completely.

GET /notifications/preferences

Get subscription configurations (comments, likes, follows, etc.).

PATCH /notifications/preferences

Enable/disable specific triggers. Accepts: likes, comments, follows, etc.
